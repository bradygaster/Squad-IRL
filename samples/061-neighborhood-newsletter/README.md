# Neighborhood Newsletter

> Create community newsletters from local happenings and announcements

## Who This Is For

🏘️ Community

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Reporter | Writer | Domain expert for neighborhood newsletter |
| Editor | Grader | Domain expert for neighborhood newsletter |
| CommunityVoice | Updater | Domain expert for neighborhood newsletter |

## How It Works

1. **Write** — Reporter reads `spec.md` and produces the first draft
2. **Grade** — Editor evaluates the output (1-100 score)
3. **Update** — If score < 90%, CommunityVoice improves the output
4. **Repeat** — Loop continues until quality threshold is met

## See It In Action

Watch the agents transform scattered neighborhood updates into a warm, professional newsletter that actually gets residents excited. From "new bakery opens Tuesday" to a complete community publication with spotlights, safety alerts, festival details, and a calendar—all with authentic neighborhood voice.
