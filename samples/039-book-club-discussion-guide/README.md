# Book Club Discussion Guide

> Create thoughtful discussion questions and themes for book club meetings

## Who This Is For

📚 Book Lovers

## The Team

| Agent | Role | What They Do |
|-------|------|-------------|
| Reader | Writer | Domain expert for book club discussion guide |
| ThemeSpotter | Grader | Domain expert for book club discussion guide |
| ConversationStarter | Updater | Domain expert for book club discussion guide |

## How It Works

1. **Write** — Reader reads `spec.md` and produces the first draft
2. **Grade** — ThemeSpotter evaluates the output (1-100 score)
3. **Update** — If score < 90%, ConversationStarter improves the output
4. **Repeat** — Loop continues until quality threshold is met

## See It In Action

Jennifer's book club discussions keep dying after 20 minutes. The agents produced 20 questions across 5 themes for *The Midnight Library*, a 90-minute meeting structure, a "Book of Regrets" icebreaker activity, and even a snack pairing suggestion — because book clubs love snacks.
