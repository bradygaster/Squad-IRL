# Social Media Content Engine

Four AI agents transform blog topics into optimized, multi-platform social media content with a weekly publishing calendar.

## Agents

| Agent | Role |
|-------|------|
| **Content Creator** | Generates 3 content variations per topic (story, data, question angles) |
| **Platform Optimizer** | Adapts each variation for Twitter, LinkedIn, and Instagram with format compliance |
| **Timing Analyst** | Analyzes simulated engagement history to recommend optimal posting times per platform |
| **Engagement Predictor** | Scores every content × platform combination and selects the top "Golden Posts" |

## What It Shows

- Multi-platform content adaptation with character-count enforcement
- Engagement heatmaps built from simulated historical data
- Weighted scoring model (hook strength, hashtag relevance, length optimization, CTA clarity, trend alignment)
- Automated weekly content calendar generation

## Run

```bash
npm install && npm start
```
